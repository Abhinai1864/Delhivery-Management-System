# 🚚 Delivery Management System

## Overview

A full-stack web application to manage shipment records using Spring Boot, React, and MySQL.

---

## Features

- Add Shipment
- View Shipments
- Edit Shipment
- Delete Shipment
- Search Shipments
- Filter by Status
- Dashboard Statistics
- Responsive UI

---

## Tech Stack

### Backend

- Java
- Spring Boot
- Spring Data JPA
- MySQL
- Swagger

### Frontend

- React
- Vite
- Axios
- Bootstrap
